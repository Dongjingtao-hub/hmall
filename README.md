# Hmall

A backend microservices project developed using Spring Cloud and Spring Boot. This project was built to practice backend development, microservice architecture, and distributed systems.
This project refactors a monolithic e-commerce application into a microservices architecture to practice distributed system design and backend development.
## Tech Stack

- Java
- Spring Boot
- Spring Cloud
- Spring Cloud Gateway
- OpenFeign
- Nacos
- MyBatis-Plus
- MySQL
- Redis
- Docker
- Maven
